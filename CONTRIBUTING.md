## Contributing

This repository contains the Akatosh node agent (Go).

Dev
- Go 1.21+
- Build: `go build ./cmd/node-agent`
- Test: `go test ./...`
- Lint (optional): `golangci-lint run`

Before PR
- Keep changes focused; explain rationale in the PR.
- Ensure builds and tests pass.
